Skill Tree

When we started working on this game, I knew that since it's an RPG, which is my favorite genre, that I wanted to make a skill tree.
I made a system where you create the tree through a json file. You have a lot of options for every node in the tree, like what stats to add/subtract, or how many times you can activate the node and how much the cost will increase for each time. On the code side you can also check stuff like how many times a node has been activated.
We decided to make the skill tree interconnected with the abilities, so every branch has uses the theme of one of the abilities, with the last node upgrading that ability.
